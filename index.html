Course We Offer

We offer Scratch, Html, Css, JavaScript and Graphic Design. Our group is focused on kids with great academic record so we can make sure they can handle both their academics and the extra lessons we will teach them.

Scratch

With Scratch, students can create their own interactive stories, games, and animations using a visual programming language. This process helps them develop creativity, systematic thinking, and collaborative skills.

HTML

HTML, also known as HyperText Markup Language, is the universal language used to structure and display web content on a browser. This coding language provides meaning and structure to the information displayed. To enhance web design, HTML is often complemented by additional technologies including Cascading Style Sheets and scripting languages such as JavaScript.

CSS

CSS, short for Cascading Style Sheets, is a language that is employed to define the style and appearance of documents written in HTML or XML. It is a fundamental technology of the internet, along with HTML and JavaScript.

JavaScript

JS, short for JavaScript, is a programming language that serves as a fundamental technology of the internet along with HTML and CSS.

Microsoft Office Suite

Microsoft Office is the collective name for Microsoft's set of home and business productivity software. The MS suite of programs includes Microsoft Word, a word processing tool; Microsoft Excel, a spreadsheet program; Microsoft PowerPoint, used for creating interactive presentations; Microsoft Outlook, used for email and calendar management; Microsoft Access, a database management program; and Microsoft OneNote, a note-taking application.

Graphic Designing

Graphic design is a field that encompasses both academic study and applied art. Its purpose is to create visual communications that convey specific messages to particular social groups, with specific goals in mind. As an interdisciplinary branch of design and fine arts, graphic design requires a diverse range of skills and knowledge.
